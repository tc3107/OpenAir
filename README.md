# OpenAir

OpenAir is an Android application. Project details and product documentation can be added here.

## Versioning
This project follows Semantic Versioning (SemVer): `MAJOR.MINOR.PATCH`.

Source of truth:
- `gradle.properties` (`VERSION_NAME`, `VERSION_CODE`)

## Build
Open the project in Android Studio and use the Gradle tasks for build/run.

## Release
Increment `VERSION_NAME` and `VERSION_CODE`, update `CHANGELOG.md`, then create a release tag.
