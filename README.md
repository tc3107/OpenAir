<h1 align="center">OpenAir</h1>


OpenAir is a simple, fast, privacyâ€‘respecting radio player and browser.

It lets you search and play live radio stations and save them to your library. 

No accounts, no ads, no telemetry, local playlists only.


## Key features

- Search radio stations by name, country, language, and genre
- Save stations to your personal library
- Wide common codec support
- Sort stations by distance to you

## Station data

Station information comes from the community-maintained  
**[Radio-Browser](https://www.radio-browser.info/)** database.

## Search

Stations can be searched by:

- Name
- Country
- Language
- Genre
